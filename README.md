## DB 설정
.env 파일을 생성해 아래와 같이 입력하여 저장해 줍니다. (app.js 와 동일한 경로에 생성)
DB_HOST = '주소'
DB_USER = '유저명'
DB_PASSWORD = '비밀번호'
DB_NAME = '데이터베이스 이름'

## 실행 방법
**1. 클론 후 의존성 설치**
**2. 'node app.js' 명령어 실행**
